## O que é OpenRouter

OpenRouter é um roteador de modelos de IA que permite acessar diversos modelos (GPT, Claude, Llama, etc.) através de uma única API.

## Por que usar

- **Modelos gratuitos** disponíveis para desenvolvimento
- **Uma API, múltiplos modelos** — troque de modelo sem mudar código
- **Fallback automático** — se um modelo está fora, outro assume

## Como começar

Basta criar uma conta, gerar uma API key e usar o endpoint compatível com OpenAI.

## Dica para devs

Use o tier gratuito para prototipagem e testes. Quando for para produção, os preços são competitivos e transparentes.
